# SAMPLES - A Portfolio Collection

This repository contains a collection of sample projects, proofs-of-concept, and other items I have worked on. Each project is self-contained in its own directory and includes its own `README.md` with more specific details.

## Projects Overview

| Project | Description |
| :--- | :--- |
| **Adv2DComponents** | A set of reusable 2D character controller components for the Godot Engine. |
| **AirControl** | A Python application for controlling a computer with hand gestures via a Leap Motion sensor. |
| **AutoAgent** | A web-based UI for an autonomous AI agent designed to solve software development tasks. |
| **LangAdventure** | An AI-powered, interactive story game for practicing Japanese language skills. |
| **LiveTranslate** | A real-time, voice-based translation tool for English and Japanese. |
| **StarterSite** | A boilerplate project for a Flask web application with common features pre-configured. |
